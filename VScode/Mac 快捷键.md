## 常用快捷键

- 格式化 Option + shift + F
- 删除行 Command + shift + K || command + del
- 查找 command + f
- 查找下一个 command + g
- 查找下一个 command + shift + g
- 向上复制 shift + option + 上
- 向下复制 shift + option + 下
- 移动行 option + 上 或 下

## 移动光标

- opt + com + 上 或 下
